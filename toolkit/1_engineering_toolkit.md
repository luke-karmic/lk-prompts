## Workflow Checklist

1. Create `docs/problem_statement.md`
2. Create `docs/design_decisions.md`
3. Go through decisions one by one
4. Architecture + Diagram
5. Gap Analysis
6. Implementation (small chunks)
7. Review + Testing
